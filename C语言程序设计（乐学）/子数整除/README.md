# 子数整除

## 题目描述

对于一个五位数a1a2a3a4a5，可将其拆分为三个子数：

- sub1=a1a2a3
- sub2=a2a3a4
- sub3=a3a4a5

例如，五位数20207可以拆分成：

- sub1=202
- sub2=020（=20）
- sub3=207

现在给定一个正整数K，要求你编程求出10000到30000之间所有满足下述条件的五位数，条件是这些五位数的三个子数sub1、sub2、sub3都可被K整除。

输出时请按照由小到大的顺序排列（每行输出一个数）。

## 样例

### 样例1

#### Input

```
15
```

#### Output

```
22555
25555
28555
30000
```
